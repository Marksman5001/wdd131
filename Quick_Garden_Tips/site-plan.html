Overview

Purpose

[Your purpose here]

The purpose of this website is to provide information about gardening and the planting of crops both indoors and outdoors, specifically to highlight the differences between the two. It should help a user get information on how to grow a plant based on the selected environment

Audience

[Your audience here]

My audience is gardeners who may need information about how to grow a plant and how it differs between growing indoor and outdoor

Dynamic elements

[Describe dynamic elements provided by Javascript here]

The dynamic elements will be on the child page, the user will select the type of plant, and whether it will be grown indoors or outdoors, and when a button is pushed, it will pull up information on that plant
